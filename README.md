
## 介绍
这是一个基于 CoffeeScript 的 NodeJS 全堆栈案例。

## 组合
* express + jade + coffee-script + mongoskin + mocha + sync

## 特点
1. 区分 MVC
  * C —— apps
  * V —— views
  * M —— models
2. 全程 CoffeeScript
3. 测试框架为 mocha

## 使用
1. clone
`git clone git://github.com/sorrycc/node-webapp-template-with-coffee.git`
2. 运行服务器
`npm run devserver`
3. 运行测试用例
`npm run test`
